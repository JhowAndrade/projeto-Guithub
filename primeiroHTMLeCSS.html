Primeiro teste para conhecer a linguagem de marcação - HTML

HTML significa Hyper Text Markup Language, onde ele é a linguagem de marcação padrão utilizado para criar páginas Web, constitido de varios elementos que informa ao navegador como exiber o conteúdo .

Conhecendo CSS linguagem de estilizar o documento HTML

CSS significa Cascading Style Sheets, descreve como os elementos HTML devem ser exibidos na tela, papel ou outra mídia, economizando muito trabalho. Ele tembém pode controlar várias páginas da web de uma só vez.
